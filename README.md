# Carte interactive de la Tunisie

Projet académique — IHEC Carthage, 2ème année Business Intelligence.

## Description

Ce projet consiste en une carte interactive de la Tunisie, réalisée dans le cadre d’un projet académique (IHEC Carthage, 2ème année Business Intelligence). La carte permet de visualiser le territoire tunisien et d’interagir avec les régions ou les villes (clic) pour afficher des informations. L’objectif est de mettre en pratique des technologies web (HTML, CSS, JavaScript) pour créer une interface claire et utilisable. Le projet peut servir de base pour afficher des données géolocalisées ou des statistiques par région.
## Fonctionnalités

-Affichage d’une carte de la Tunisie avec les 24 gouvernorats

-Zones cliquables définies à l’aide de balises <map> et <area>

-Affichage dynamique des informations de chaque région

-Affichage d’images associées aux gouvernorats

-Interaction utilisateur sans rechargement de la page
## Langages

- HTML, CSS, JavaScript

## Objectif

Mettre en pratique les bases du développement web, notamment la structuration d’une page HTML, la mise en forme avec CSS et l’ajout d’interactivité via JavaScript.
Ce projet permet également de mieux comprendre la gestion des événements utilisateur et la manipulation dynamique du contenu d’une page web.
